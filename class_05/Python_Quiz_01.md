**1. Which of the following is a valid variable name in Python?**

- a) 2name
- b) name_2
- c) name-2
- d) @name

**2. What is the output of the following code?**

```python
a = 5
b = 2
print(a * b)
```

- a) 7
- b) 10
- c) 52
- d) 2.5

**3. Which data type is used to store a collection of items in order and is mutable (Changeable)?**

- a) tuple
- b) list

**4. What will be the output of this code?**

```python
fruits = ["apple", "banana", "mango"]
print(fruits[1])
```

- a) apple
- b) banana
- c) mango
- d) 1

**5. Which function is used to take input from the user in Python?**

- a) input()

- d) read()

**6. What is the correct way to define a function in Python?**

- a) function greet():

**7. What is the output of the following code?**

```python
student = {"name": "Ali", "age": 15}
print(student["age"])
```

- a) name
- b) Ali
- c) 15
- d) age

**8. Which of the following is an immutable (Unchangeable) data structure?**

- a) list
- b) dictionary
- c) tuple
- d) set

**9. What will this code print?**

```python
for i in range(1, 4):
    print(i)
```

- a) 1 2 3
- b) 0 1 2 3
- c) 1 2 3 4
- d) 0 1 2

**10. What is the output of this code?**

```python
x = 3

    print("Yes")
else:
    print("No")
```

- a) Yes
- b) No
- c) Error
- d) 3
